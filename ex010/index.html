Usando link externos

Aqui eu posso usar um link para que as pessoas possam ir direto para uma página que eu queira, ex: acesse o link do GitHub do Guanabara

Ou o meu canal do youtube que não presta mais pra nada.😕

Eu posso usar um link externo deixando ele aparecer em cima da minha página principal como esse aqui.Porém a minha página principal vai sumir isso só com a tag (a href) ex: testando o primeiro link, que vai direto pra página do pdf de estudo.

Ou eu posso usar a mesma tag (a href) colar o link e finalizar com a tag (target="blank") é indicado usar também juntas a tag(rel="external") para mostrar pro navegador que você está indo para o site externo.Depois do link copiado, antes do (a/) de finalizar a tag indo para uma página externa e deixando a minha página principal aberta 😎.

criei uma página 2 e para ir pra lá eu preciso colocar (o nome do meu arquivo que está dentro da mesma pasta)

Eu também posso ultilizar a tag (real="next") dentro do link da minha página ex: a href="pag2.html antes de fechar eu adiciono a tag rel="next". Assim eu estou avisando ao mecanismo de busca do google qual é a minha próxima página eu não deixo ele decidir qual é a página e para ele voltar para a página anterior (real="prev") eu aviso pra ele qual é a página anterior.

ir para a segunda página

Eu posso adicionar uma terceira página ou quantas eu quiser, mais se estiver dentro de uma pasta preciso usar o nome da pasta para achar o arquivo Terceira página

Caso eu tenha que fazer referencia de um site que sei se é fake ou qual a intenção dele ali ou saber um aval do site ex: um site de fofoca eu uso a tag (a href= colo o link target="blank" rel= nofollow ) e o (rel="external" eu tamém aviso pro google que eu estou saindo da página) pra não ficar indexando ou ficar seguindo a página. ele abre a página mais eu aviso para o google que eu não dou o aval para esse site eu não conheço ele.
